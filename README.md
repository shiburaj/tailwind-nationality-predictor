# Nationality Predictor

[DEMO](index.html)

A smart app to predict the nationality of a person by their name.

### Endpoints / API used
- [Nationality API](https://api.nationalize.io/?name=tom)
- [Country to Name API](https://restcountries.com/v3.1/alpha/IN)


### Library Used
- Tailwind for UI
- VueJS
- ReactJS
